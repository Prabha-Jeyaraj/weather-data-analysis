# Weather Data Analysis

## Overview
Analyzed a real-world weather dataset to explore temperature, 
humidity and rainfall trends across cities using Python.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Source: Kaggle (Szeged Weather Dataset)
- Records: 96,453 weather readings

## Key Findings
- Temperature peaks in summer months and drops in winter
- Humidity remains consistently high throughout the year
- Wind speed varies moderately across months

## Project Structure
- Weather_Data_Analysis.ipynb → Main analysis notebook
- weatherHistory.csv → Dataset
