> Exploring real-world weather patterns through data analysis and visualization

## 📌 Overview
This project analyzes a real-world weather dataset containing **96,453 weather records** to uncover seasonal patterns, temperature trends, humidity distribution, and wind speed variations across cities using Python data science libraries.

---

## 📊 Visualizations & Insights

### 🌡️ Temperature Trends
- Analyzed average monthly temperature across all years
- Clear seasonal pattern observed — **peaks in July, drops in January**
- Line chart used to visualize smooth temperature curve

### 💧 Humidity Distribution
- Explored how humidity values are spread across the dataset
- Over **60% of days recorded high humidity (above 0.7)**
- Histogram with KDE curve used for distribution analysis

### 🌬️ Wind Speed by Month
- Compared average wind speed across all 12 months
- **Winter months recorded highest wind speeds**
- Bar chart used for easy month-by-month comparison

### 🗺️ Weather Feature Heatmap
- Compared Temperature, Humidity and Wind Speed together
- Monthly patterns visible across all 3 features simultaneously
- Heatmap used for multi-feature comparison

---

## 🔍 Key Findings
✅ Temperature **peaks in summer** and **drops in winter** showing strong seasonality

✅ Humidity is **consistently high** throughout the year (above 0.7 on most days)

✅ Wind speed is **highest in winter months** and lowest in summer

✅ All three weather features show **clear monthly patterns**
